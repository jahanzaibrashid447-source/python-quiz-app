# Python Quiz App

A command-line quiz application built using Python.  
The app asks multiple questions, checks user answers, and shows the final score.

## Features
- Multiple choice questions
- Answer validation
- Score calculation
- Simple and user-friendly interface

## Technologies Used
- Python
- File Handling
- Conditional Statements
- Loops

## How to Run

1. Download the project
2. Open terminal in project folder
3. Run:

python quiz.py
